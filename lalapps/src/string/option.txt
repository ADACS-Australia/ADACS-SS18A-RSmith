-- Chi2 selection cuts --
Format :
IFO  PAR[0]  PAR[1]  PAR[2]

An event is cut if :
log10(chi2) > PAR[0] 
&& log10(chi2) > PAR[1] * log10(SNR) + PAR[2]

The cut is not applied if this file is missing
or if one IFO is missing
-------------------------
L1  0.5  1.8  -2.3
H1  0.5  1.8  -2.4
H2  0.5  1.8  -2.4
V1  0.5  1.8  -2.3
